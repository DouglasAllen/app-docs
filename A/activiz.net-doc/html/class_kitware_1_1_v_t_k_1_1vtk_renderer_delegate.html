Detailed Description

vtkRendererDelegate - Render the props of a vtkRenderer

Description vtkRendererDelegate is an abstract class with a pure virtual method Render. This method replaces the Render method of vtkRenderer to allow custom rendering from an external project. A RendererDelegate is connected to a vtkRenderer with method SetDelegate(). An external project just has to provide a concrete implementation of vtkRendererDelegate.
