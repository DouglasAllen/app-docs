Detailed Description

vtkXMLPStructuredGridWriter - Write PVTK XML StructuredGrid files.

Description vtkXMLPStructuredGridWriter writes the PVTK XML StructuredGrid file format. One structured grid input can be written into a parallel file format with any number of pieces spread across files. The standard extension for this writer's file format is "pvts". This writer uses vtkXMLStructuredGridWriter to write the individual piece files.
