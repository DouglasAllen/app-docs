Detailed Description

vtkXMLCompositeDataWriter - Writer for multi-group datasets

Description vtkXMLCompositeDataWriter writes (serially) the VTK XML multi-group, multi-block hierarchical and hierarchical box files. XML multi-group data files are meta-files that point to a list of serial VTK XML files.
