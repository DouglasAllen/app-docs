Detailed Description

vtkTypeUInt8Array - dynamic, self-adjusting array of vtkTypeUInt8

Description vtkTypeUInt8Array is an array of values of type vtkTypeUInt8. It provides methods for insertion and retrieval of values and will automatically resize itself to hold new data.
