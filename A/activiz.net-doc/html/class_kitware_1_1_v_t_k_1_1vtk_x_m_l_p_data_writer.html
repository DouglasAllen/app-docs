Detailed Description

vtkXMLPDataWriter - Write data in a parallel XML format.

Description vtkXMLPDataWriter is the superclass for all XML parallel data set writers. It provides functionality needed for writing parallel formats, such as the selection of which writer writes the summary file and what range of pieces are assigned to each serial writer.
