Detailed Description

vtkUnstructuredGridReader - read vtk unstructured grid data file

Description vtkUnstructuredGridReader is a source object that reads ASCII or binary unstructured grid data files in vtk format. (see text for format details). The output of this reader is a single vtkUnstructuredGrid data object. The superclass of this class, vtkDataReader, provides many methods for controlling the reading of the data file, see vtkDataReader for more information. Caveats Binary files written on one system may not be readable on other systems.
