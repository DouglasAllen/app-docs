Detailed Description

vtkExtractPiece

Description vtkExtractPiece returns the appropriate piece of each sub-dataset in the vtkCompositeDataSet. This filter can handle sub-datasets of type vtkImageData, vtkPolyData, vtkRectilinearGrid, vtkStructuredGrid, and vtkUnstructuredGrid; it does not handle sub-grids of type vtkCompositeDataSet.
