Detailed Description

vtkTypeUInt64Array - dynamic, self-adjusting array of vtkTypeUInt64

Description vtkTypeUInt64Array is an array of values of type vtkTypeUInt64. It provides methods for insertion and retrieval of values and will automatically resize itself to hold new data.
