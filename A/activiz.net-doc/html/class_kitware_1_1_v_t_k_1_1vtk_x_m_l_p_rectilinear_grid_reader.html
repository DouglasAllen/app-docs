Detailed Description

vtkXMLPRectilinearGridReader - Read PVTK XML RectilinearGrid files.

Description vtkXMLPRectilinearGridReader reads the PVTK XML RectilinearGrid file format. This reads the parallel format's summary file and then uses vtkXMLRectilinearGridReader to read data from the individual RectilinearGrid piece files. Streaming is supported. The standard extension for this reader's file format is "pvtr".
