Detailed Description

vtkXMLStructuredGridWriter - Write VTK XML StructuredGrid files.

Description vtkXMLStructuredGridWriter writes the VTK XML StructuredGrid file format. One structured grid input can be written into one file in any number of streamed pieces. The standard extension for this writer's file format is "vts". This writer is also used to write a single piece of the parallel file format.
