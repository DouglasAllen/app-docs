Detailed Description

vtkStandardPolyDataPainter - A standard implementation of vtkPolyDataPainter.

Description vtkStandardPolyDataPainter is a catch-all painter. It should work with pretty much any vtkPolyData, and attributes, and vtkPolyDataPainterDeviceAdapter. On the flip side, the vtkStandardPolyDataPainter will be slower than the more special purpose painters. Thanks Support for generic vertex attributes in VTK was contributed in collaboration with Stephane Ploix at EDF.
