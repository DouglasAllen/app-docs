Detailed Description

vtkPMultiCorrelativeStatistics - A class for parallel bivariate correlative statistics

Description vtkPMultiCorrelativeStatistics is vtkMultiCorrelativeStatistics subclass for parallel datasets. It learns and derives the global statistical model on each node, but assesses each individual data points on the node that owns it.
