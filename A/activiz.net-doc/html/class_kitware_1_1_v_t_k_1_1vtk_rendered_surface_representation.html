Detailed Description

vtkRenderedSurfaceRepresentation - Displays a geometric dataset as a surface.

Description vtkRenderedSurfaceRepresentation is used to show a geometric dataset in a view. The representation uses a vtkGeometryFilter to convert the dataset to polygonal data (e.g. volumetric data is converted to its external surface). The representation may then be added to vtkRenderView.
