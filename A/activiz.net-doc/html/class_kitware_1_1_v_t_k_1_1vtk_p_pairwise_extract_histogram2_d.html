Detailed Description

vtkPPairwiseExtractHistogram2D - compute a 2D histogram between all adjacent columns of an input vtkTable in parallel.

Description This class does exactly the same this as vtkPairwiseExtractHistogram2D, but does it in a multi-process environment. After each node computes their own local histograms, this class does an AllReduce that distributes the sum of all local histograms onto each node.

Because vtkPairwiseExtractHistogram2D is a light wrapper around a series of vtkExtractHistogram2D classes, this class just overrides the function that instantiates new histogram filters and returns the parallel version (vtkPExtractHistogram2D).
