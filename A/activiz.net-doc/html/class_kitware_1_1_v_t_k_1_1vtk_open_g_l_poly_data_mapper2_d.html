Detailed Description

vtkOpenGLPolyDataMapper2D - 2D PolyData support for OpenGL

Description vtkOpenGLPolyDataMapper2D provides 2D PolyData annotation support for vtk under OpenGL. Normally the user should use vtkPolyDataMapper2D which in turn will use this class.
