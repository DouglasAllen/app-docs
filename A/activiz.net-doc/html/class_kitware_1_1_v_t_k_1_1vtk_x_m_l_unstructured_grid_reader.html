Detailed Description

vtkXMLUnstructuredGridReader - Read VTK XML UnstructuredGrid files.

Description vtkXMLUnstructuredGridReader reads the VTK XML UnstructuredGrid file format. One unstructured grid file can be read to produce one output. Streaming is supported. The standard extension for this reader's file format is "vtu". This reader is also used to read a single piece of the parallel file format.
