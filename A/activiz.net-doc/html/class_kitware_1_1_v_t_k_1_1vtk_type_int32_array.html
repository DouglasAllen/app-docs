Detailed Description

vtkTypeInt32Array - dynamic, self-adjusting array of vtkTypeInt32

Description vtkTypeInt32Array is an array of values of type vtkTypeInt32. It provides methods for insertion and retrieval of values and will automatically resize itself to hold new data.
