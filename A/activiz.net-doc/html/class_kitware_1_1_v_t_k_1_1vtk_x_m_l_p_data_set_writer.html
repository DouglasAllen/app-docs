Detailed Description

vtkXMLPDataSetWriter - Write any type of PVTK XML file.

Description vtkXMLPDataSetWriter is a wrapper around the PVTK XML file format writers. Given an input vtkDataSet, the correct writer is automatically selected based on the type of input.
