Detailed Description

vtkGraphReader - read vtkGraph data file

Description vtkGraphReader is a source object that reads ASCII or binary vtkGraph data files in vtk format. (see text for format details). The output of this reader is a single vtkGraph data object. The superclass of this class, vtkDataReader, provides many methods for controlling the reading of the data file, see vtkDataReader for more information. Caveats Binary files written on one system may not be readable on other systems.
