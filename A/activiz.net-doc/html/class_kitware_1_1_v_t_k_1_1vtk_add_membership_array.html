Detailed Description

vtkAddMembershipArray - Add an array to the output indicating membership within an input selection.

Description This filter takes an input selection, vtkDataSetAttribute information, and data object and adds a bit array to the output vtkDataSetAttributes indicating whether each index was selected or not.
