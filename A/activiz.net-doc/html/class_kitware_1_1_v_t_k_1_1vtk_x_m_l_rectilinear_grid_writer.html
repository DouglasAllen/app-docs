Detailed Description

vtkXMLRectilinearGridWriter - Write VTK XML RectilinearGrid files.

Description vtkXMLRectilinearGridWriter writes the VTK XML RectilinearGrid file format. One rectilinear grid input can be written into one file in any number of streamed pieces. The standard extension for this writer's file format is "vtr". This writer is also used to write a single piece of the parallel file format.
