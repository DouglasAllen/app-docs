Detailed Description

vtkXMLRectilinearGridReader - Read VTK XML RectilinearGrid files.

Description vtkXMLRectilinearGridReader reads the VTK XML RectilinearGrid file format. One rectilinear grid file can be read to produce one output. Streaming is supported. The standard extension for this reader's file format is "vtr". This reader is also used to read a single piece of the parallel file format.
