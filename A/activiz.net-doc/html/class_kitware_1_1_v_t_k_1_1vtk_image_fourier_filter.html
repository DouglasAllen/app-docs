Detailed Description

vtkImageFourierFilter - Superclass that implements complex numbers.

Description vtkImageFourierFilter is a class of filters that use complex numbers this superclass is a container for methods that manipulate these structure including fast Fourier transforms. Complex numbers may become a class. This should really be a helper class.
