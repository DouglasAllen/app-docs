Detailed Description

vtkStructuredPointsWriter - write vtk structured points data file

Description vtkStructuredPointsWriter is a source object that writes ASCII or binary structured points data in vtk file format. See text for format details. Caveats Binary files written on one system may not be readable on other systems.
