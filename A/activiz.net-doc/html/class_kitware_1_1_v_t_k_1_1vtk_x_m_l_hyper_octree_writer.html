Detailed Description

vtkXMLHyperOctreeWriter - Write VTK XML HyperOctree files.

Description vtkXMLHyperOctreeWriter writes the VTK XML HyperOctree file format. One HyperOctree input can be written into one file in any number of streamed pieces. The standard extension for this writer's file format is "vto". This writer is also used to write a single piece of the parallel file format.
