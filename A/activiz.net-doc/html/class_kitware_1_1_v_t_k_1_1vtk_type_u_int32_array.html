Detailed Description

vtkTypeUInt32Array - dynamic, self-adjusting array of vtkTypeUInt32

Description vtkTypeUInt32Array is an array of values of type vtkTypeUInt32. It provides methods for insertion and retrieval of values and will automatically resize itself to hold new data.
