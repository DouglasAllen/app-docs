Detailed Description

vtkTypeInt8Array - dynamic, self-adjusting array of vtkTypeInt8

Description vtkTypeInt8Array is an array of values of type vtkTypeInt8. It provides methods for insertion and retrieval of values and will automatically resize itself to hold new data.
