Detailed Description

vtkCompositePolyDataMapper2 - mapper for composite dataset consisting of polygonal data.

Description vtkCompositePolyDataMapper2 is similar to vtkCompositePolyDataMapper except that instead of creating individual mapper for each block in the composite dataset, it iterates over the blocks internally.
