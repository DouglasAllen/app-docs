Detailed Description

vtkCylindricalTransform - cylindrical to rectangular coords and back

Description vtkCylindricalTransform will convert (r,theta,z) coordinates to (x,y,z) coordinates and back again. The angles are given in radians. By default, it converts cylindrical coordinates to rectangular, but GetInverse() returns a transform that will do the opposite. The equation that is used is x = r*cos(theta), y = r*sin(theta), z = z. Caveats This transform is not well behaved along the line x=y=0 (i.e. along the z-axis)
