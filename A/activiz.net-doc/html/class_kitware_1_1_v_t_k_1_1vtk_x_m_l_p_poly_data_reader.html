Detailed Description

vtkXMLPPolyDataReader - Read PVTK XML PolyData files.

Description vtkXMLPPolyDataReader reads the PVTK XML PolyData file format. This reads the parallel format's summary file and then uses vtkXMLPolyDataReader to read data from the individual PolyData piece files. Streaming is supported. The standard extension for this reader's file format is "pvtp".
