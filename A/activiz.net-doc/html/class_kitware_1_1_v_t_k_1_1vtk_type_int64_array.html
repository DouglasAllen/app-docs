Detailed Description

vtkTypeInt64Array - dynamic, self-adjusting array of vtkTypeInt64

Description vtkTypeInt64Array is an array of values of type vtkTypeInt64. It provides methods for insertion and retrieval of values and will automatically resize itself to hold new data.
