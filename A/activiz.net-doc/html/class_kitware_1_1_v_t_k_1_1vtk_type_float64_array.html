Detailed Description

vtkTypeFloat64Array - dynamic, self-adjusting array of vtkTypeFloat64

Description vtkTypeFloat64Array is an array of values of type vtkTypeFloat64. It provides methods for insertion and retrieval of values and will automatically resize itself to hold new data.
