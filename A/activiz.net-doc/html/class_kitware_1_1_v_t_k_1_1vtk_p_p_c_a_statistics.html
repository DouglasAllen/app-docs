Detailed Description

vtkPPCAStatistics - A class for parallel principal component analysis

Description vtkPPCAStatistics is vtkPCAStatistics subclass for parallel datasets. It learns and derives the global statistical model on each node, but assesses each individual data points on the node that owns it.
