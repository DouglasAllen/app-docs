Detailed Description

vtkPComputeHistogram2DOutliers - extract outlier rows from a vtkTable based on input 2D histograms, in parallel.

Description This class does exactly the same this as vtkComputeHistogram2DOutliers, but does it in a multi-process environment. After each node computes their own local outliers, class does an AllGather that distributes the outliers to every node. This could probably just be a Gather onto the root node instead.

After this operation, the row selection will only contain local row ids, since I'm not sure how to deal with distributed ids.
