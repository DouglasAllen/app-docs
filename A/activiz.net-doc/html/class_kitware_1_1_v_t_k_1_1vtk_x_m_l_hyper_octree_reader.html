Detailed Description

vtkXMLHyperOctreeReader - Read VTK XML HyperOctree files.

Description vtkXMLHyperOctreeReader reads the VTK XML HyperOctree file format. One rectilinear grid file can be read to produce one output. Streaming is supported. The standard extension for this reader's file format is "vto". This reader is also used to read a single piece of the parallel file format.
