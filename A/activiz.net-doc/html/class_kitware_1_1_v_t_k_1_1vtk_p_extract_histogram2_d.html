Detailed Description

vtkPExtractHistogram2D - compute a 2D histogram between two columns of an input vtkTable in parallel.

Description This class does exactly the same this as vtkExtractHistogram2D, but does it in a multi-process environment. After each node computes their own local histograms, this class does an AllReduce that distributes the sum of all local histograms onto each node.

Thanks Developed by David Feng and Philippe Pebay at Sandia National Laboratories ------------------------------------------------------------------------------
