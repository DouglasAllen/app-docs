Detailed Description

vtkTypeInt16Array - dynamic, self-adjusting array of vtkTypeInt16

Description vtkTypeInt16Array is an array of values of type vtkTypeInt16. It provides methods for insertion and retrieval of values and will automatically resize itself to hold new data.
