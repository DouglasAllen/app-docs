Detailed Description

vtkTableToArray - converts a vtkTable to a matrix.

Description Converts a vtkTable into a dense matrix. Use AddColumn() to designate one-to-many table columns that will become columns in the output matrix.a

Using AddColumn() it is possible to duplicate / reorder columns in arbitrary ways.

Caveats Only produces vtkDenseArray<double>, regardless of the input table column types.

Thanks Developed by Timothy M. Shead (tshead@sandia.gov) at Sandia National Laboratories.
