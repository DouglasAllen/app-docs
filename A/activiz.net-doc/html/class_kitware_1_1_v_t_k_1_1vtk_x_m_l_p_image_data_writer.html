Detailed Description

vtkXMLPImageDataWriter - Write PVTK XML ImageData files.

Description vtkXMLPImageDataWriter writes the PVTK XML ImageData file format. One image data input can be written into a parallel file format with any number of pieces spread across files. The standard extension for this writer's file format is "pvti". This writer uses vtkXMLImageDataWriter to write the individual piece files.
