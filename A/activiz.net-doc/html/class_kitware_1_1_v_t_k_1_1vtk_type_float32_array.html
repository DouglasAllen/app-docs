Detailed Description

vtkTypeFloat32Array - dynamic, self-adjusting array of vtkTypeFloat32

Description vtkTypeFloat32Array is an array of values of type vtkTypeFloat32. It provides methods for insertion and retrieval of values and will automatically resize itself to hold new data.
