Synopsis¶

New in version 1.4.

Set system’s hostname Currently implemented on Debian, Ubuntu, Fedora, RedHat, openSUSE, Linaro, ScientificLinux, Arch, CentOS, AMI.

Options¶

parameter required default choices comments
name yes

    Requires hostname
