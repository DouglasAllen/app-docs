Ns_DbSpReturnCode

Overview

Get return code from a stored procedure

Syntax

    int Ns_DbSpReturnCode(
    Ns_DbHandle *handle, 
    char *returnCode, 
    int bufsize
    );

Description

Get the return code from a stored procedure after running Ns_DbSpExec. Returns NS_OK on success, NS_ERROR on failure.
