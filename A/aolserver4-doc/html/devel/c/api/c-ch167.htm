Ns_DiffTime

Overview

Get difference between two times

Syntax

    void Ns_DiffTime (
    Ns_Time* t1, 
    Ns_Time* t2, 
    Ns_Time* result
    );

Description

Determine the difference in seconds between two Ns_Time structures. The result is put into result.
