Ns_ConnCopyToChannel

Overview

Copy content to Tcl channel

Syntax

    int Ns_ConnCopyToChannel (
    Ns_Conn* conn, 
    size_t iToCopy, 
    Tcl_Channel chan
    );

Description

Copy content, such as in a PUT request, from the connection into an open Tcl channel. iToCopy bytes will be copied.
