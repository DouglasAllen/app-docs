Ns_CacheCreateEntry

Overview

Create a cache entry

Syntax

    Ns_Entry* Ns_CacheCreateEntry (
    Ns_Cache* cache, 
    char* key, 
    int* pnew
    );

Description

Create a new cache entry in the specified cache. This function emulates Tcl_CreateHashEntry's interface.
