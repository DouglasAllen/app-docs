Ns_InitializeSemaphore

Overview

Initialize a semaphore

Syntax

    int Ns_InitializeSemaphore(
    Ns_Semaphore * sema, 
    int beg_count
    );

Description

Initialize the semaphore with a semaphore count of beg_count.

Ns_SemaInit is the preferred function for initializing a semaphore.
