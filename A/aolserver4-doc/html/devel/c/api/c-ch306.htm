Ns_RWLockRdLock

Overview

Acquire a read lock

Syntax

    void Ns_RWLockRdLock (
    Ns_RWLock *lockPtr 
    );

Description

Ns_RWLockRdLock acquires a read lock. Any number of read locks can be pending. If there's a write lock active, the read lock acquisition blocks until the write lock is released.
