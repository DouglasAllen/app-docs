Ns_GetSockAddr

Overview

Get socket driver address

Syntax

    int Ns_GetSockAddr (
    struct sockaddr_in* saPtr, 
    char* host, 
    int port
    );

Description

Specify host and port in saPtr. Specify an IP address or a hostname for host. A NULL host means INADDR_ANY.
