Ns_SetUpdate

Overview

Update an Ns_Set value

Syntax

    void Ns_SetUpdate (
    Ns_Set* set, 
    char* key, 
    char* value
    );

Description

Remove an item from the Ns_Set whose key = key, if one exists, and then re-add the item with the new value.
