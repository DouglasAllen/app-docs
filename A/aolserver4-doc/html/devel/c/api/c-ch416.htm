Ns_UnlockMutex

Overview

Unlock the mutual exclusion lock

Syntax

    int Ns_UnlockMutex(
    Ns_Mutex * mutex
    );

Description

Unlock the mutex.

Ns_MutexLock is the preferred function for unlocking a mutex.
