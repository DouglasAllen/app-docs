Ns_TclAppendInt

Overview

Append integer to Tcl result

Syntax

    void Ns_TclAppendInt (
    Tcl_Interp* interp, 
    int value 
    );

Description

Append an integer to the Tcl result. This is essentially a safe version of sprintf(interp->result, "%d", value).
