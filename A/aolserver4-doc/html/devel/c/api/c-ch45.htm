Ns_CondBroadcast

Overview

Wake up all threads waiting on a cond

Syntax

    void Ns_CondBroadcast (
    Ns_Cond* 
    );

Description

Wake up all threads that are waiting on a cond. For more informations on condition variables, look at the pthread_cond_wait(3P) man page.
