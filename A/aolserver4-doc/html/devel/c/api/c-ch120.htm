Ns_ConnSetHeaders

Overview

Set the value for a header field

Syntax

    void Ns_ConnSetHeaders(
    Ns_Conn *conn,
    char *field,
    char *value
    );

Description

The Ns_ConnSetHeaders function sets the value of a field in the output headers, replacing an existing field/value pair.
