Ns_DestroySemaphore

Overview

Free a semaphore's resources

Syntax

    int Ns_DestroySemaphore(
    Ns_Semaphore * sema
    );

Description

Free the resources associated with the semaphore.

Ns_SemaDestroy is the preferred function for freeing a semaphore's resources.
