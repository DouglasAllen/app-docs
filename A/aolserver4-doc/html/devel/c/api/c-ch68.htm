Ns_ConnCopyToFd

Overview

Copy content to file descriptor

Syntax

    int Ns_ConnCopyToFd (
    Ns_Conn* conn, 
    size_t iToCopy, 
    int fd
    );

Description

Copy content, such as in a PUT request, from the connection into an open file descriptor. iToCopy bytes will be copied.
