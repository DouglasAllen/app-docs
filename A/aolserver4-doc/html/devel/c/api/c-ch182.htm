Ns_Encrypt

Overview

Encrypt key

Syntax

    char *Ns_Encrypt(
    char *key,
    char *salt,
    char buf[]
    );

Description

The Ns_Encrypt function encrypts the specified key, perturbed by salt. The result is returned in buf, which should be at least NS_ENCRYPT_BUFSIZE bytes in size.
