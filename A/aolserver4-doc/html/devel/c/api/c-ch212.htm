Ns_GetUserHome

Overview

Get UNIX user's home directory

Syntax

    int Ns_GetUserHome(
    Ns_DString *pds,
    char *user
    );

Description

The Ns_GetUserHome function returns NS_TRUE and appends the user's home directory to the Ns_DString passed in, or it returns NS_FALSE if the user doesn't exist.
