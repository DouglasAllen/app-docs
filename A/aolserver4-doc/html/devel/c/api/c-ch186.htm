Ns_ExecProc

Overview

Execute file with argument array

Syntax

    int Ns_ExecProc (
    char* sExec, 
    char** argv
    );

Description

Execute the file sExec. All input will come from stdin. All output will go to stdout.

The argv is a null-terminated array of argument strings, like this: { "foo", "bar", NULL }.
