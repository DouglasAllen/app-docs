Ns_QuoteHtml

Overview

Quote an HTML string

Syntax

    void Ns_QuoteHtml(
    Ns_DString *pds,
    char *string
    );

Description

The Ns_QuoteHtml function appends the given string to the Ns_DString, making the following substitutions that allow HTML to be included in another HTML page as plain text:

<

&lt;

>

&gt;

&

&amp;
