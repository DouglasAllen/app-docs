Ns_CondTimedWait

Overview

Block on a cond

Syntax

    int Ns_CondTimedWait (
    Ns_Cond *condPtr , 
    Ns_Mutex *mutexPtr , 
    Ns_Time *timePtr 
    );

Description

Block on a cond until signaled or the specified time expires. The time is absolute. The Ns_Time value can be manipulated with Ns_GetTime, Ns_DiffTime, or Ns_IncrTime.
