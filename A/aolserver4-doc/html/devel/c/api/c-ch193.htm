Ns_Free

Overview

Free a block of allocated memory

Syntax

    void *Ns_Free(
    void *buf
    );

Description

The Ns_Free function frees any memory allocated by the Ns_Malloc, Ns_Calloc, or Ns_Realloc functions. This function replaces the system free function.
