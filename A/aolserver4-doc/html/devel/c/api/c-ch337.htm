Ns_SetListFree

Overview

Free a list of sets

Syntax

    void Ns_SetListFree(
    Ns_Set **sets
    );

Description

The Ns_SetListFree function frees the sets in a NULL-terminated array of sets and then frees the array itself. This function is normally used to free a list of sets generated by Ns_SetSplit.
