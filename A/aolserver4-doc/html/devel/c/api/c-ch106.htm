Ns_ConnReturnOk

Overview

Return an "OK" HTTP status line.

Syntax

    int Ns_ConnReturnOk(
    Ns_Conn *conn
    );

Description

Calls Ns_ConnReturnStatus or Ns_ConnReturnNotice with a status code of 200 to indicate that the request was successful.
