Ns_SetUserAuthorizeProc

Overview

Set function used by Ns_AuthorizeUser

Syntax

    void Ns_SetUserAuthorizeProc(
    Ns_UserAuthorizeProc *procPtr
    );

Description

Sets a procedure to handle calls to Ns_AuthorizeUser. This function should only be called once per execution of AOLserver. The procPtr should be of the form:

    typedef int (Ns_UserAuthorizeProc) (char *user, char *passwd);

It should return NS_OK on a match or NS_ERROR if any problem is encountered or the password does not match.
