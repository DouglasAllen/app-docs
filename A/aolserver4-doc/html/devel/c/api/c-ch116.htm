Ns_ConnSendFd

Overview

Write file to connection content

Syntax

    int Ns_ConnSendFd(
    Ns_Conn *conn,
    int fd,
    int len
    );

Description

The Ns_ConnSendFd function writes len bytes from the file pointed to by fd to the connection. Ns_ConnSendFd returns the status NS_ERROR or NS_OK.
