Ns_PoolAlloc

Overview

Allocate memory within a pool

Syntax

    void* Ns_PoolAlloc (
    Ns_Pool* pool, 
    unsigned int size
    );

Description

Alloc memory within a pool. Memory pools are thread-specific memory that reduce the number of globally-locking malloc(3C) calls.
