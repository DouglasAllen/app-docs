Ns_AbsTimedWaitForEvent

Overview

Wait for an event to be broadcast

Syntax

    int Ns_AbsTimedWaitForEvent (
    Ns_Cond* event, 
    Ns_Mutex* lock, 
    time_t abstime
    );

Description

Wait for an event to be broadcast or the current time to be abstime, whichever comes first.
