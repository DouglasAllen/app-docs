Ns_SetMerge

Overview

Merge two sets

Syntax

    void Ns_SetMerge(
    Ns_Set *high,
    Ns_Set *low
    );

Description

The Ns_SetMerge function appends any fields from the low set to the high set if a field with the name key name does not already exist in the high set.
