Example: Simple ADP Template

  • Returning the time and browser is now a bit cleaner:

    <%
    set headers [ns_conn headers]
    set browser [ns_set iget $headers User-Agent]
    set time [clock seconds]
    %>
    <html><body>
    Time: <%= $time %>
    Browser: <%= $browser %>
    </body></html>
