  • Asus p535 (through web interface)
  • Nokia - N800 (through web interface)
  • E28 E2831 (through web interface)
  • HTC 3450 and Diamond Touch (through web interface)
  • Motorola V220 (in AT-mode through the cable connection)
  • Motorola Z6 (in Server mode through IP over bluetooth connection)
  • AVer Media Studio 507 (IR remote RM-F7)

  • Sony Ericsson phones
