8.5 geometry

This module, written by Philippe Ivaldi, provides an extensive set of geometry routines, including perpendicular symbols and a triangle structure. Link to the documentation for the geometry module are posted here: http://asymptote.sourceforge.net/links.html, including an extensive set of examples, http://www.piprime.fr/files/asymptote/geometry/, and an index:
