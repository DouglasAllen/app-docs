8.19 MetaPost

This package provides some useful routines to help MetaPost users migrate old MetaPost code to Asymptote. Further contributions here are welcome.

Unlike MetaPost, Asymptote does not implicitly solve linear equations and therefore does not have the notion of a whatever unknown. The routine extension (see extension) provides a useful replacement for a common use of whatever: finding the intersection point of the lines through P, Q and p, q. For less common occurrences of whatever, one can use the built-in explicit linear equation solver solve instead.
