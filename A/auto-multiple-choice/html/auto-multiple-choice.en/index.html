Abstract

AMC is a set of utilities allowing the use of MCQ written in plain text or LaTeX, and their automated correction and grading from scans of the answer sheets using Optical Mark Recognition (OMR). This document describes its use (for the version 1.1.1~svn 1104).

The AMC project is hosted by Gna ! : you can find the AMC web-site there http://home.gna.org/auto-qcm/, and the web-page where the project is hosted there https://gna.org/projects/auto-qcm/.
