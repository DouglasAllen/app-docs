rollbar_deployment - Notify Rollbar about app deployments¶

Author:Max Riveiro
  • Synopsis
  • Options
  • Examples

Synopsis¶

New in version 1.6.

Notify Rollbar about app deployments (see https://rollbar.com/docs/deploys_other/)

Options¶

parameter required default choices comments
comment no
    		Deploy comment (e.g. what is being deployed).
    environment yes
      		Name of the environment being deployed, e.g. 'production'.
      revision yes
        		Revision number/sha being deployed.
        rollbar_user no
          		Rollbar username of the user who deployed.
          token yes
            		Your project access token.
            url no https://api.rollbar.com/api/1/deploy/
              		Optional URL to submit the notification to.
              user no
                		User who deployed.
                validate_certs no yes
                • yes
                • no
                		 If no, SSL certificates for the target url will not be validated. This should only be used on personally controlled sites using self-signed certificates.

                Examples¶


                - rollbar_deployment: token=AAAAAA
                      environment='staging'
                      user='ansible'
                      revision=4.2,
                      rollbar_user='admin',
                      comment='Test Deploy'
